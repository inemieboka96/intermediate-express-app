User & Products API

Modern Express.js API for managing users and products with style.

Express.js Docs Test with Postman

Overview

This API provides endpoints to manage users and products. Built with Express.js, it's perfect for learning RESTful API concepts, prototyping, or as a foundation for more advanced applications.

View API Endpoints

See all available API endpoints on the new API Endpoints page.

Endpoints

Users API

  • GET /api/users - List all users
  • GET /api/users/:id - Get a user by ID
  • POST /api/users - Create a new user
  • PUT /api/users/:id - Update a user
  • DELETE /api/users/:id - Delete a user

Products API

  • GET /api/products - List all products
  • GET /api/products/:id - Get a product by ID
  • POST /api/products - Create a new product
  • PUT /api/products/:id - Update a product
  • DELETE /api/products/:id - Delete a product

Getting Started

  1. Clone the repository and install dependencies.
  2. Start the server: node app.js
  3. Use a tool like Postman or your browser to interact with the endpoints above.
